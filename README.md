# ida-mappings
An IDA plugin for mappings for function names and arg names.

Rename list format:
```renamelist.txt
sub_xxxxxx fn_name arg1 arg2
sub_xxxxxx fn_name arg1 arg2
...
```

Instruction:
1. Download script and run in ida
2. Select file with mappings
3. Set additional offset if you want
4. Done (afaik)
